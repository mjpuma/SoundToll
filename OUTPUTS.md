## Map Figures

### soundtoll_map_zoom.png, soundtoll_map_wide.png

**Caption:** Sound Toll shipping network, 1700–1720. Directed port-to-port routes through the Danish Sound. Node size ∝ total passages (in + out); edge width ∝ route traffic. Lambert Conformal projection (central lon 12°, lat 57°); geodesic (great-circle) paths. Gdansk highlighted. Top 40 ports by passages labeled. Zoom: North Sea–Baltic only (lon −2 to 26, lat 52–64). Wide: includes Spanish ports (lon −15 to 40, lat 35–70). Routes with <5 passages excluded. Legend shows visible range only. Data: STRO.

**Scaling:** Node size = 8 + (p − min_p) / (max_p − min_p) × 20 points; edge width = max(0.4, w / max_weight × 3 × 0.35). Legend uses min/max from ports within extent.

## Period Comparison

### period_comparison.png

**Caption:** Pre-plague (1705–1708) vs post-plague (1710–1713) network comparison. Panels: ports (nodes), routes (edges), total passages, density, reciprocity (fraction of bidirectional routes), avg. betweenness centrality. Summary table: period, nodes, edges, passages, % total, density, reciprocity, betweenness. Data: STRO.

**Computations:** Density = m / (n×(n−1)); reciprocity = nx.reciprocity(G); betweenness = nx.betweenness_centrality(G, weight="weight").

---

### period_comparison.csv

**Columns:** period, n_nodes, n_edges, density, total_passages, pct_passages, max_route_passages, reciprocity, avg_betweenness, top_ports.
